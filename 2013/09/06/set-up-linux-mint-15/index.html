安裝 Linux Mint 15 & Windows 7 雙作業系統

前言

以前在使用Linux的作業系統時，都是將Ubuntu裝在一台很舊的PC上，使用上倒也沒有什麼地方可以挑剔。近來Mint的崛起讓我有點心動想要跳槽過去。

使用起來是覺得Mint的GUI更加直覺好用。因為老舊的PC即將壽終正寢，我就把Mint裝在我的Notebook上，但又想保留Windows 7所以做了雙系統的配置，以下本篇將記錄安裝的過程。

目前採用Linux Mint 15 Olivia Desktop版本則是使用Cinnamon，搭配Windows 7雙系統。

前置準備作業

  • 4G以上 USB 一支
  • 已經安裝好Windows 7 筆電一台

1. Windows 7壓縮磁碟

說明：首先要在Windows 7的C槽底下壓縮一塊空間，這塊空間將會是你的Linux所擁有的大小。

操作：進入「我的電腦」的「磁碟管理」在C槽那按右鍵選擇「壓縮磁碟區」，在跳出的視窗找「輸入要壓縮的空間大小」這裡就是Linux將佔有的空間，請輸入你所規劃的空間大小，完成後點擊下面的「壓縮」。

2. Linux Mint 15 映像檔轉可開機的Live USB

說明：以往安裝Linux都需要把ISO燒錄進CD做Live CD，不過此方法既不環保，也不夠快速。這裡推薦一個軟體UNetbootin它能將Linux的ISO檔弄進去USB裡面做成Live USB。

操作：下載後直接開啓程式，選「映像檔」挑選已經下載好的Linux Mint 15 ISO檔，按下確定後等待完成即可。

3. Linux Mint硬碟分割

說明：做硬碟分割的時候跟一般Linux一樣，主要分割出4個區塊

  • Swap 磁區: Swap 磁區功能是使用硬碟空間來作為虛擬記憶體，但現今記憶體(RAM)十分便宜，隨處都能看到4G以上的RAM，所以這塊區域可以規劃小一點。至於到底要設多大?請參考Ubuntu 官方幫助而目前我是設定1GB。

  • /root: 根目錄又做”/“是整個系統最重要的一個目錄，所有的目錄皆是根目錄衍生而來，裡面的資料為系統開機時需要特定的開機軟體、核心檔案、開機所需程式、 函式庫等等檔案資料。通常不需要設很大，因為應用程式不建議安裝在這裡，保持根目錄小可以使系效能更加。建議配置15G~20G。

  • /boot: 這個目錄主要在放置開機會使用到的檔案，包括Linux核心檔案以及開機選單與開機所需設定檔等等。不需要設定很大，建議配置500MB~1000MB。

  • /home: 這是系統預設的使用者家目錄，用來放個人資料與應用程式的地方，建議此項最後配置，所剩空間全部歸給他。

操作：將電腦關機後先插上USB，開機後進入BIOS修改開機順序，選擇USB啟動儲存後離開，系統將重新啟動進入Linux Mint 15 的桌面後，雙擊桌面Install Linux Mint後開始安裝，選擇語言後開始分割系統硬碟。

當出現分配硬碟空間時，請選第三個選項”Something else”，然後選擇Free space點選下方的「新增」接著開始按照上述的說明分配硬碟空間。

注意以下幾點:

  • Type of new partition: Swap磁區空間請選擇Logical，其他的選為Primary

  • New partiotion size in MB: 輸入個磁區想要分配的大小，可參考上述的說明

  • Location for the new partition(磁碟區的位置): 都是選beginning

  • Use as: 除Swap選擇SWAP其他則選Ext4

  • Mount point(掛載點): 裡面有”/“, “/boot”, “/home”，而Swap不需要選

    全部設定完後，就可以選擇Install Now，大致上就算完成了。
